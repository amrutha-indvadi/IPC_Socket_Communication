# TCP Client-Server Communication
A simple TCP client-server communication project developed in C using the Winsock API. This project demonstrates the fundamentals of socket programming, network communication, and client-server architecture.

## Features
* TCP socket communication
* Client-server architecture
* Bidirectional message exchange
* Real-time message transmission
* Winsock API implementation

## Concepts Demonstrated
* Socket Creation
* Socket Binding
* Listening for Connections
* Accepting Client Requests
* Establishing TCP Connections
* Sending and Receiving Messages
* Inter-Process Communication (IPC)

## How to Run

### Server
1. Compile the server program.
2. Run the executable.
3. The server will listen for incoming client connections on port 8888.

### Client
1. Update the server IP address in the client source code.
2. Compile the client program.
3. Run the executable.
4. Connect to the server and exchange messages.
